# Nolli OS
A simple, light-hearted Bootloader/OS

## Dependencies
+ nasm
+ clang
+ ld (needs to be changed in the build script per user)
+ dd
+ qemu

1. ```./build.sh```
2. ```qemu-system-x86_64 nolli.bin```
3. Sheer joy
